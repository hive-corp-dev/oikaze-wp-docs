基本的には`rem`指定ではなく`leading-none`から`leading-loose`までのクラスを使用する。

サイト全体のデフォルトの行間は、`leading-normal`に設定する。

もしデフォルトの行間が`1.5`以外の場合は、`tailwind.config.js`の`theme.extend.lineHeight`から上書きする。

## カスタム行間の設定方法

行間の上書き及びカスタム行間の追加は、`tailwind.config.js`の`theme.extend.lineHeight`に記述する。

export const configCode = `
  module.exports = {
    theme: {
      extend: {
        lineHeight: {
          normal: "1.6", // デフォルトの行間を1.6に上書き
          wide: "2.25", // カスタム行間を追加
        },
      },
    },
  };
`;
