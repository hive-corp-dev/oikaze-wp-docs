プロジェクトに色を設定する方法、使用する方法について。

## 色の設定方法

色を追加する場合は、`theme.json`の`"palette"`に記述する。

- `name`: 色の名前
- `slug`: 色のスラッグ（識別子）
- `color`: 色の値（16 進数カラーコード）

import { Code } from "@astrojs/starlight/components";
import { Aside } from "@astrojs/starlight/components";
import DevCaution from "../../../components/DevCaution.astro";

export const exampleCode = `
{
  "settings": {
    "color": {
      "palette": [
        {
          "name": "Base",
          "slug": "base",
          "color": "#4C331F"
        },
        {
          "name": "ブラック",
          "slug": "black",
          "color": "#000000"
        },
        {
          "name": "メインカラー",
          "slug": "main",
          "color": "#3d50f5"
        },
      ],
    }
  }
}
`;
export const fileName = "app/themes/theme/theme.json";

## 色の使用方法

色を使用するには、`theme.json`の`"palette"`に記述した`"slug"`を使用する。

- `bg-{slug}`: 背景色
- `text-{slug}`: 文字色
- `border-{slug}`: ボーダーの色

export const exampleHtml = `
<div class="bg-main">背景色はメインカラーです。</div>

<p class="text-main">文字色はメインカラーです。</p>

<div class="border border-main">ボーダーはメインカラーです。</div>
`;
